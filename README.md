# Power BI – Data Jobs Dashboards

This repository contains two Power BI dashboards built from a 2024 data jobs postings dataset.  
The goal is to explore salaries, skills demand, job types, and trends for data roles.

## Dashboards

### 1. Data_Jobs_Dashboard_v1 (Drill-through version)

Folder: [`Data_Jobs_Dashboard_v1`](./Data_Jobs_Dashboard_v1)

- Main summary page with KPIs and visuals
- Drill-through page for job title–level details
- Screenshots in `images/`
- See its own [`README`](./Data_Jobs_Dashboard_v1/README.md) for full description

### 2. Data_Jobs_Dashboard_v2 (Single-page version)

Folder: [`Data_Jobs_Dashboard_v2`](./Data_Jobs_Dashboard_v2)

- Single-page, simplified dashboard for quick insight
- Focused KPIs and interactive visuals
- Screenshots in `images/`
- See its own [`README`](./Data_Jobs_Dashboard_v2/README.md) for full description

## Files

- `Data_Jobs_Dashboard_v1/*.pbix` – Power BI file for v1
- `Data_Jobs_Dashboard_v2/*.pbix` – Power BI file for v2
